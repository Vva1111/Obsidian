##### Scripts
- **"dev":"vite"**
	- Vite 개발 서버를 실행
	- 개발 중에 적용하는 애플리케이션 변경 사항을 빠르게 확인, 테스트하기 위해 사용, HMR을 통해 코드 변경이 실시간으로 변경되는 것을 확인하기 위해 사용한다
	- 예시) npm run dev
- **"serve":"vite preview"**
	- Vite로 빌드된 적정 파일을 제공하는 로컬 서버 실행
	- 프로덕션 빌드된 애플리케이션을 로컬에서 미리 보기 위해 사용한다. 실제 배포된 상태에서의 동작을 확인할 수 있음
	- 예시) npm run serve
- 위 두 명령어는 각각 개발 중/배포 후의 애플리케이션 상태를 확인하는데 사용된다.
