##### 1. 초기화 ( Initialization )
- Vue3 컴포넌트가 생성되는 초기 단계
- Setup 함수 호출
- reactive data properties 초기화 및 메모리 할당
- 이벤트 리스너 설정 및 메모리에 참조로 이벤트 리스너 큐에 저장

##### 2. 마운트 ( Mounting )
- Setup 함수에서 반환된 객체는 Vue3에 의해 추적되며, 이 단계에서는 실제 DOM에 컴포넌트가 마운트된다.
- 위 과정에서 메모리에는 컴포넌트 인스턴스와 관련된 데이터와 메소드의 참조가 저장된다.

##### 3. 업데이트 ( Updating )
- 컴포넌트의 데이터 변경시 Vue3는 반응성 시스템을 통해 이를 감지하고 컴포넌트를 업데이트한다.
- 이 단계에서는 변경된 데이터가 메모리에서 덮어씌워진다
- 필요에 따라 새로운 메모리 공간이 예약될 수 있다.
- 변경사항이 반영된 새로운 가상돔이 생성, 이전의 가상돔과 비교해 실제 돔에 반영된다.

##### 4. 파괴 ( Unmounting )
- 컴포넌트가 돔에서 제거될 때
- 이 단계에서는 컴포넌트 인스턴스와 그에 연결된 모든 데이터, 메소드, 이벤트 리스너들이 메모리에서 해제되어 가비지 컬렉션의 대상이 된다.
