### find(GPT)

```bash
find [검색할 디렉토리] [옵션] [검색 조건]
```

1. **현재 디렉토리에서 모든 파일 찾기**:   
    ```bash
    find . -type f
    ```
2. **특정 이름의 파일 찾기**:
    ```bash
    find /path/to/directory -name "filename.txt"
    ```
3. **확장자가 `.txt`인 모든 파일 찾기**
    ```bash
    find /path/to/directory -type f -name "*.txt"
    ```
4. **특정 크기 이상의 파일 찾기**:
    ```bash
    find /path/to/directory -type f -size +100M
    ```
5. **수정 시간이 7일 이내인 파일 찾기**:
    ```bash
    find /path/to/directory -type f -mtime -7
    ```
6. **찾은 파일 삭제하기**:
    ```bash
    find /path/to/directory -type f -name "*.tmp" -delete
    ```
7. **특정 사용자 소유의 파일 찾기**:
    ```bash
    find /path/to/directory -type f -user username
    ```
8. **디렉토리만 찾기**:
    ```bash
    find /path/to/directory -type d
    ```

#### 파일 이동(GPT)
```bash
mv [옵션] <원본 파일 또는 디렉토리> <대상 경로>
```

1. **파일 이동**:   
    ```bash
    mv /path/to/source/file.txt /path/to/destination/
    ```
2. **파일 이름 변경**:
    ```bash
    mv /path/to/file.txt /path/to/newfile.txt
    ```
3. **디렉토리 이동**:
    ```bash
    mv /path/to/source_directory /path/to/destination/
    ```
4. **덮어쓰기 확인**:
    ```bash
    mv -i /path/to/source/file.txt /path/to/destination/
    ```
    `-i` 옵션은 대상에 동일한 이름의 파일이 있을 경우 덮어쓸지 확인합니다.
5. **강제로 덮어쓰기**:
    ```bash
    mv -f /path/to/source/file.txt /path/to/destination/
    ```
    `-f` 옵션은 확인 없이 강제로 덮어씌웁니다.
