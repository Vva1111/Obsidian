### Tip.
- apt-get install vim 을 사용해 설치가 정상적으로 이루어지지 않는 경우, apt-get을 update 해볼 필요가 있다.

#### 모드 전환
- i :  삽입 모드
- esc : 명령 모드, 키 입력이 명령으로 해석
- v : 비주얼 모드, 텍스트 선택 및 수정

#### 이동
- 왼(h), 아래(j), 위(k), 오(l)
- gg : 파일의 첫 번째 줄로 이동
- G : 파일의 마지막 줄로 이동
- 0 : 현재 줄의 시작으로 이동
- $ : 현재 줄의 끝으로 이동

#### 편집
- u : 최근 명령을 취소
- dd : 현재 줄을 삭제
- yy : 현재 줄을 복사
- p : 클립보드에 있는 내용을 현재 커서 위치에 붙여 넣음
- x : 커서가 가리키는 문자를 삭제

#### 저장 및 종료
- :w - 변경 내용을 저장
- :q - Vim 종료
- :wq 또는 :x - 변경 내용을 저장하고 Vim 종료
- :q! - 변경 내용을 저장하지 않고 Vim 강제 종료
