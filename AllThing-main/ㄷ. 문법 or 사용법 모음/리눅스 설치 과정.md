<h4>3. 설치 </h4>
1. USB ( OS 마운트된 USB ) 연결

3. 파티션 설정 ( ex )
	- LVM ( home ) : Boot 기본 파티션
	- root : 나머지 모두
	- swap : 메모리 절반 정도
