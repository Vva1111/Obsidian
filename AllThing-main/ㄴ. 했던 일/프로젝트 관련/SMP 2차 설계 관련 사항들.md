##### 초안
- 자원 정보(CPU, 메모리, Storage 등등)는 정해진 카테고리 내에서 선택
	- 제일 높은 사양 이상을 원할 시는 max 범위 내에서 직접 선택해 설정할 수 있도록 선택 팝업 제공
- 정해진 도메인 내에서 선택 사용 ( 기본 : cocoa.icu )
- 사용자에게 입력 받아야 하는 필수 API
	- 사용자 등록
	- 사용자 현황
	- 사용자 삭제
	- 사용자 갱신
	- Health Check
	- SaaS Default Information
	- ...
- DB 접근 정보 등 추가 입력 정보를 위한 팝업 창 제공
- yaml 직접 설정을 위한 yaml editor 및 설정 입력 변환 기능 필요



##### 01. 31 회의 이후 정해진 사항
- 소프트웨어 등록과 설치는 분리
	- 등록은 기본 정보(이름, 설명, 아이콘 정도) 단순 등록
	- 이후 등록된 소프트웨어를 CMP에 설치
- 소프트웨어 목록에서 현재 소프트웨어의 상태(등록, 설치, 설치 오류 등) 표시
- 대시보드(소프트웨어 현황)는 소프트웨어 리스트에 함께 표시되어야 할 수도 있고, 그건 정해야함
- 소프트웨어 관리는 별개의 메뉴로 root 에서 1depth 로 들어가야 한다
