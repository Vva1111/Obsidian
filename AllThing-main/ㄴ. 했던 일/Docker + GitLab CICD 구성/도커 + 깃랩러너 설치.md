##### 1. 도커 설치
```sh
sudo wget -qO- http://get.docker.com/ | sh
```

##### 2. 도커 명령어 사용을 위한 도커 권한 그룹에 현재 사용자추가
`ssh 재연결 해야 권한 추가됨`
```sh
sudo usermod -aG docker $USER
```

##### 3. 도커허브에서 깃랩러너 이미지 다운로드
```sh
docker pull gitlab/gitlab-runner:latest
```

##### 4. gitlab-runner 이미지 실행
```sh
$ docker run -d --name gitlab-runner --restart always -v /data/runner/config:/etc/gitlab-runner -v /var/run/docker.sock:/var/run/docker.sock -e TZ=Asia/Seoul gitlab/gitlab-runner:latest
```

##### 5. gitlab에 runner 등록 (gitlab.yescnc-cloud.com 의 러너 등록 방법 내용 참조, url과 token은 각 gilab 페이지를 참조해서 적용)
```sh
$ docker exec -it gitlab-runner gitlab-runner register --url https://gitlab.yescnc-cloud.com/ --registration-token GR1348941YujiWqVWL8HEtu4DgRiL
```

#### 6. gitlab-runner 구성 변경
```sh
$ sudo nano /data/runner/config/config.toml
```

수정 및 추가해줘야 할 부분
```toml
..
[[runners]]
..
  [runners.docker]
    volumes = ["/var/run/docker.sock:/var/run/docker.sock", "/cache"]
    allowed_pull_policies = ["if-not-present", "always"]
    pull_policy = "if-not-present"
    ..    
```
