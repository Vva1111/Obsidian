- **GIT 계정 생성**

- **GIT에서 사용할 SSH 키 쌍 생성 (로컬에서 생성 후 git에 키 값 저장)**
```sh
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```
- SSH 프로토콜을 사용하기 위한 새로운 공개/비공개 키 쌍을 생성하는데 사용
- **ssh-keygen** : SSH 키를 생성, 관리하는데 사용하는 명령
- **-t rsa** : 키의 타입을 지정한다. rsa는 가장 널리 사용되는 공개 키 암호화 알고리즘 중 하나
- **-b 4096** : 비트 수 지정. 4096은 키의 크기를 의미하며, 보안성을 높이기 위해 4096 비트를 사용한다. 기본 값은 2048 이지만, 더 높은 수치를 선택해 보안을 강화할 수 있다.
- **-C "이메일"** : 키에 대한 주석 추가. 키를 식별하는데 도움이 되며, 주로 생성한 사용자의 이메일을 주석으로 사용한다.

- **ssh 키 디렉토리 구분으로 파악하는게 제일 편할 듯**
